Esta tarea consiste en un registro para Instagram utilizando HTML, CSS y PHP. Se utilizó XAMPP para la conexión y PostgreSQL para la creación de la base de datos.

El login se encuentra en el archivo ig.php, todas las imágenes que se ocupan se encuentran en la carpeta fuente.

La conexión con la base de datos se realiza en el archivo conexion.php, la cual se deben reemplazar los datos segun su configuración de bd.
